# QuickNotes

• Notas curtas
• Guardadas em JSON no DataStore
• UI minimalista

🌟 Mini-Hackathon (2 aulas × 4h): "Smart Micro-App Challenge"

O objetivo e criar uma micro-aplicacao Android funcional, que demonstre:
• dominio de Jetpack Compose
• gestao de estado
• uso de layouts
• persistencia com DataStore (Preferences ou Proto)

🧩 Tema Geral
Criar uma micro-app que resolva um problema simples do dia a dia, com:
• Varios ecras
• UI em Compose
• Estado reativo
• Persistencia com DataStore

## O que a app faz
- Lista principal de notas curtas (foco total no essencial).
- Persistencia em DataStore com JSON.
- 2 ecras com navegacao rapida por setas no fundo.
- Dark mode com toggle e estado guardado.
- Splash minimalista de entrada.

## Ecras
1. `SplashActivity`
2. `MainActivity`
3. `SecondScreenActivity`

## Estrutura principal
- `app/src/main/java/com/notone/protodatastore/UserPreferences.kt`
- `app/src/main/java/com/notone/protodatastore/MainActivity.kt`
- `app/src/main/java/com/notone/protodatastore/SecondScreenActivity.kt`
- `app/src/main/java/com/notone/protodatastore/SplashActivity.kt`
